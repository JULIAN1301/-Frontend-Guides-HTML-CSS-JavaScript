📏 1. Unidades de Medida en CSS

Las unidades de medida en CSS permiten definir tamaños, márgenes, anchos y más. Se dividen en relativas y absolutas.

📜 Lista de unidades de medida en CSS

px, cm, mm, in, pt, pc, %, em, rem, vw, vh, vmin, vmax, ex, ch

📌 1.1. Unidades Absolutas

Las unidades absolutas tienen un tamaño fijo y no cambian según el contexto.

  • px - Píxeles (unidad más común en pantallas).
  • cm, mm, in - Centímetros, milímetros y pulgadas.
  • pt, pc - Puntos y picas (usados en impresión).
div { width: 100px; height: 50px; }
100px × 50px

📐 1.2. Unidades Relativas

Las unidades relativas dependen del tamaño del contenedor o del texto.

  • % - Relativo al contenedor padre.
  • em - Relativo al tamaño de fuente del elemento.
  • rem - Relativo al tamaño de fuente de html.
  • vw, vh - Porcentaje del ancho o alto de la ventana.
div { width: 50%; height: 10vh; font-size: 2em; }
50% de ancho, 10vh de alto

🎯 1.3. Comparación entre em y rem

em escala según el elemento padre, mientras que rem se basa en html.

html { font-size: 16px; }
.container { font-size: 20px; }
.em-example { font-size: 2em; } /* 40px */
.rem-example { font-size: 2rem; } /* 32px (16px × 2) */

🎨 2. Colores y Fondos en CSS

CSS permite definir colores y fondos para los elementos de una página web.

📜 Lista de propiedades de colores y fondos en CSS

color, background-color, background-image, background-repeat, background-position, 
background-size, background-attachment, background-clip, background-origin, 
background-blend-mode, opacity, linear-gradient, radial-gradient

🌈 2.1. Propiedad color

Define el color del texto de un elemento. Se pueden usar:

  • Colores por nombre: Ejemplo: red, blue, green.
  • Valores HEX: Ejemplo: #ff0000 (rojo), #00ff00 (verde).
  • RGB: Ejemplo: rgb(255, 0, 0) (rojo).
  • HSL: Ejemplo: hsl(0, 100%, 50%) (rojo).
p { color: blue; }

Este texto es azul

🖼️ 2.2. Propiedad background-color

Define el color de fondo de un elemento.

div { background-color: #f0f0f0; }
Este es un fondo gris claro

📷 2.3. Imágenes de fondo con background-image

Se pueden usar imágenes como fondo.

body { background-image: url('Visual_icon.svg.png'); }
Imagen de fondo

🔄 2.4. Control de repetición con background-repeat

Define cómo se repite la imagen de fondo.

  • repeat (predeterminado)
  • no-repeat (sin repetición)
  • repeat-x (solo horizontal)
  • repeat-y (solo vertical)
div { background-repeat: no-repeat; }
Fondo sin repetición

📏 2.5. Posición del fondo con background-position

Permite alinear la imagen de fondo dentro del elemento.

div { background-position: center center; }
Fondo centrado

🔳 2.6. Tamaño del fondo con background-size

Define el tamaño de la imagen de fondo.

  • auto: Tamaño original.
  • cover: Ajusta la imagen para cubrir el área.
  • contain: Ajusta la imagen sin recortarla.
div { background-size: cover; }
Fondo con contain

🎨 2.7. Fondos degradados con background y linear-gradient

CSS permite crear degradados como fondos.

div { background: linear-gradient(to right, red, blue); }
Fondo con degradado

🔤 3. Tipografías y Estilos de Texto en CSS

CSS permite personalizar las fuentes y la presentación del texto en una página web.

📜 Lista de propiedades de tipografía en CSS

font-family, font-size, font-style, font-weight, letter-spacing, line-height, 
text-align, text-decoration, text-transform, text-indent, white-space, text-shadow

📌 3.1. Propiedad font-family

Define la fuente del texto. Se recomienda incluir fuentes alternativas.

p { font-family: Arial, sans-serif; }

Este texto usa Arial.

🔠 3.2. Tamaño del texto con font-size

Define el tamaño del texto en píxeles (px), em o rem.

h1 { font-size: 32px; }

Texto de 32px

📝 3.3. Estilos de texto con font-style y font-weight

Permiten modificar la apariencia del texto.

  • font-style: normal, italic, oblique.
  • font-weight: normal, bold, valores numéricos (100-900).
p { font-style: italic; font-weight: bold; }

Texto en negrita e itálica

📏 3.4. Espaciado con letter-spacing y line-height

Controla la separación entre letras y líneas de texto.

p { letter-spacing: 2px; line-height: 1.5; }

Este texto tiene más espacio entre letras y líneas.

📍 3.5. Alineación del texto con text-align

Define la alineación del texto dentro de un contenedor.

p { text-align: center; }

Texto centrado

✨ 3.6. Decoración con text-decoration

Se usa para subrayar, tachar o modificar el estilo del texto.

a { text-decoration: none; }

Enlace sin subrayado

🌟 3.7. Sombras en el texto con text-shadow

Permite agregar sombras y efectos al texto.

h1 { text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.5); }

Texto con sombra

📏 4. Espaciado y Dimensiones en CSS

CSS permite definir el tamaño de los elementos y controlar su espaciado.

📜 Lista de propiedades de espaciado y dimensiones en CSS

margin, padding, width, height, max-width, min-width, max-height, min-height, 
box-sizing, overflow, display, visibility

🛑 4.1. Margen exterior con margin

Controla el espacio entre un elemento y otros elementos.

div { margin: 20px; }
Este div tiene un margen de 20px

📦 4.2. Espaciado interno con padding

Define el espacio dentro del borde del elemento.

div { padding: 15px; }
Este div tiene un padding de 15px

📏 4.3. Ancho y alto con width y height

Establece el tamaño de un elemento.

div { width: 300px; height: 200px; }
300x200px

⚠️ 4.4. Tamaño máximo y mínimo con max-width, min-width, max-height, min-height

Controla las dimensiones mínimas y máximas de un elemento.

div { max-width: 500px; min-height: 100px; }
Este div tiene un máximo de 500px de ancho y un mínimo de 100px de alto

📦 4.5. Modelo de caja con box-sizing

Determina cómo se calculan las dimensiones de un elemento.

div { box-sizing: border-box; }
Este div usa box-sizing: border-box

🛠️ 4.6. Control de desbordamiento con overflow

Define qué sucede cuando el contenido de un elemento es más grande que su contenedor.

  • visible: El contenido sobresale.
  • hidden: Se oculta el contenido desbordado.
  • scroll: Agrega barras de desplazamiento.
  • auto: Agrega barras de desplazamiento si es necesario.
div { overflow: auto; }
Este div tiene contenido que sobrepasa el tamaño, por lo que se mostrará un scroll si es necesario. Sigue escribiendo para probar el desbordamiento y ver cómo se comporta con diferentes valores de overflow.

📌 5. Posicionamiento y Display en CSS

CSS permite controlar la ubicación y el comportamiento de los elementos en la página.

📜 Lista de propiedades de posicionamiento y display en CSS

position, top, right, bottom, left, z-index, display, float, clear, visibility, 
overflow, flex, justify-content, align-items, grid, grid-template-columns

📍 5.1. Tipos de posicionamiento con position

Define cómo se coloca un elemento en la página.

  • static: Posición por defecto.
  • relative: Se posiciona en relación con su posición original.
  • absolute: Se posiciona en relación con su contenedor más cercano con position: relative.
  • fixed: Se mantiene fijo en la pantalla, sin importar el desplazamiento.
  • sticky: Se comporta como relative hasta alcanzar un punto de la pantalla y se fija.
div { position: absolute; top: 50px; left: 100px; }

📏 5.2. Control de visibilidad con z-index

Define qué elemento está por encima de otro en el eje Z.

div { position: absolute; z-index: 10; }
Este div tiene z-index 10

🖼️ 5.3. Control de visualización con display

Determina cómo se comporta un elemento en la página.

  • block: Ocupa toda la anchura disponible.
  • inline: Solo ocupa el espacio del contenido.
  • inline-block: Como inline, pero permite definir ancho y alto.
  • none: Oculta el elemento sin eliminarlo del DOM.
span { display: inline-block; width: 100px; }
Span con inline-block

🔄 5.4. Alineación con float y clear

Permiten alinear elementos dentro de un contenedor.

img { float: left; margin-right: 10px; }
Este texto rodea la imagen flotante.

📦 5.5. Control de flexibilidad con flex

Usado para distribuir elementos en una fila o columna de manera flexible.

div { display: flex; justify-content: center; align-items: center; }
Centrado con flexbox

📐 5.6. Diseño de cuadrículas con grid

Permite organizar los elementos en filas y columnas.

div { display: grid; grid-template-columns: repeat(3, 1fr); }

🎭 6. Transformaciones y Transiciones en CSS

CSS permite modificar la apariencia de los elementos con transformaciones y animaciones suaves con transiciones.

📜 Lista de propiedades de transformaciones y transiciones en CSS

transform, rotate, scale, skew, translate, transition, transition-property, 
transition-duration, transition-timing-function, transition-delay

🔄 6.1. Transformaciones con transform

Permiten cambiar la escala, rotación, inclinación o posición de un elemento.

  • rotate(deg): Rota el elemento.
  • scale(x, y): Escala el elemento.
  • skew(x, y): Inclina el elemento.
  • translate(x, y): Mueve el elemento en el eje X e Y.
div { transform: rotate(45deg); }
Rotado

🎨 6.2. Transiciones con transition

Permiten animaciones suaves cuando cambia una propiedad.

button { transition: background-color 0.5s ease-in-out; }

⏳ 6.3. Propiedades de transition

  • transition-property: Propiedad a animar (ej. opacity).
  • transition-duration: Duración de la animación.
  • transition-timing-function: Velocidad de la animación (ej. ease, linear).
  • transition-delay: Retraso antes de iniciar.
div { transition: all 0.3s ease-in-out; }
Pasa el mouse

✨ 6.4. Combinación de transformaciones y transiciones

Podemos combinar ambos efectos para crear animaciones fluidas.

div { 
        transform: scale(1);
        transition: transform 0.5s ease;
    }
    
    div:hover {
        transform: scale(1.2);
    }
Escalar

🎭 7. Pseudoclases y Pseudoelementos en CSS

Las pseudoclases y pseudoelementos permiten aplicar estilos dinámicos y manipular partes específicas de un elemento.

📜 Lista de pseudoclases y pseudoelementos en CSS

:hover, :focus, :nth-child(n), :first-child, :last-child, :not(selector),
::before, ::after, ::first-letter, ::first-line, ::selection

🔹 7.1. Pseudoclases

Las pseudoclases aplican estilos a un elemento en un estado específico.

  • :hover: Aplica estilo cuando el usuario pasa el mouse.
  • :focus: Se activa cuando un elemento recibe foco.
  • :nth-child(n): Aplica estilo al enésimo hijo de un contenedor.
  • :first-child y :last-child: Seleccionan el primer o último hijo.
  • :not(selector): Excluye elementos que coincidan con el selector.
button:hover { background-color: blue; }

🎨 7.2. Pseudoelementos

Los pseudoelementos permiten aplicar estilos a partes específicas de un elemento.

  • ::before: Inserta contenido antes del elemento.
  • ::after: Inserta contenido después del elemento.
  • ::first-letter: Aplica estilo a la primera letra.
  • ::first-line: Aplica estilo a la primera línea de un párrafo.
  • ::selection: Define el estilo del texto seleccionado por el usuario.
p::first-letter { font-size: 2em; font-weight: bold; }

Ejemplo de pseudoelemento ::first-letter aplicado.

🛠️ 7.3. Ejemplo combinado

Podemos combinar pseudoclases y pseudoelementos para obtener efectos avanzados.

a:hover::after { content: ' 🔗'; color: red; }
Enlace interactivo

🌈 8. Variables CSS y Custom Properties

Las variables CSS (también llamadas Custom Properties) permiten almacenar valores reutilizables en toda la hoja de estilos.

📜 Lista de propiedades relacionadas con variables CSS

:root, --nombre-variable, var(), setProperty()

🎯 8.1. Declaración de Variables

Las variables CSS se definen con --nombre-variable dentro de :root para usarlas globalmente.

:root {
        --color-primario: #3498db;
        --tamano-texto: 16px;
    }
Este div usa una variable CSS

🎨 8.2. Uso de Variables

Para usar una variable, se emplea la función var().

h1 {
        color: var(--color-primario);
        font-size: var(--tamano-texto);
    }

Título con variable CSS

🔄 8.3. Variables en Elementos Específicos

Las variables también pueden definirse dentro de elementos específicos.

button {
        --btn-color: #e74c3c;
        background-color: var(--btn-color);
    }

🛠️ 8.4. Valores Predeterminados

Si una variable no está definida, var() permite asignar un valor por defecto.

p {
        color: var(--color-secundario, #2ecc71);
    }

Este párrafo usa un valor predeterminado

⚡ 8.5. Variables Dinámicas con JavaScript

Las variables CSS pueden modificarse dinámicamente con JavaScript.

document.documentElement.style.setProperty('--color-primario', 'purple');

📱 9. Responsive Design en CSS

El diseño responsive permite adaptar la apariencia de una página a diferentes dispositivos y tamaños de pantalla.

📜 Lista de técnicas para diseño responsive en CSS

@media, flex-wrap, grid-template-columns, vw, vh, max-width, min-width, viewport

📏 9.1. Uso de Media Queries

Las @media queries permiten aplicar estilos según el ancho de la pantalla.

@media (max-width: 768px) {
        body {
            background-color: lightgray;
        }
    }
Redimensiona la ventana para ver el efecto

🔹 9.2. Unidades Relativas

Usar em, rem, vw y vh en lugar de px facilita la adaptación.

h1 {
        font-size: 5vw; /* Tamaño relativo al ancho de la pantalla */
    }

Este título cambia de tamaño según el ancho de la pantalla

📐 9.3. Diseño Flexible con Flexbox

Flexbox facilita la adaptación de los elementos.

.contenedor {
        display: flex;
        flex-wrap: wrap;
        justify-content: center;
    }

Grid Layout permite reorganizar elementos según el tamaño de pantalla.

.grid {

        grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

📌 9.5. Imágenes Responsivas

Las imágenes pueden ajustarse automáticamente al tamaño del contenedor.

img {
        max-width: 100%;
        height: auto;
    }

📳 9.6. Meta Viewport

El siguiente meta permite que la página se adapte a dispositivos móviles.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Este meta ayuda a escalar correctamente el diseño en dispositivos móviles.
